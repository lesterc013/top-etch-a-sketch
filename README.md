# top-etch-a-sketch

The Odin Project's project (lol) to "flex your DOM manipulation skills" and also practice CSS concepts like flexbox.

So far, I've learnt how to

- Make good use of flexbox: Flexbox is a 1D based styling tool. As such, if using it to create a grid, we should have to use at least 2 flexboxes; Rows that holds the cells, that each stacks on top of each other
- Make use of event bubbling instead of having individual event listeners. Ties in nicely with what I've learnt so far to keep logic centralised using event pub/subs
